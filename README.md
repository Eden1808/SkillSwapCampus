# ğŸ“± Overview

SkillSwap Campus is a student-to-student skill exchange Android application.

Instead of paying for courses, students exchange knowledge.

The app allows students to:

* Offer skills they already have
* Request skills they want to learn
* Approve or reject collaboration requests
* Create real academic matches

The system is powered by Firebase Realtime Database and built entirely in Java.
